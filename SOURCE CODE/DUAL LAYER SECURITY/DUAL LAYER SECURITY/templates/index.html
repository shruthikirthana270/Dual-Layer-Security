Dual-Layer Security System

Welcome to Dual-Layer Security System

This project provides a secure data sharing mechanism using a dual-layer protection model that combines Cryptography and Steganography. It ensures that even if the data is intercepted, it remains unreadable and hidden within an image.

About the Project

The system allows two main roles: Owner and User. The Owner uploads files, which are first encrypted using cryptographic algorithms and then hidden inside an image using steganography. The User can request access to these files, and upon the Ownerâ€™s approval, securely download and decrypt them.
